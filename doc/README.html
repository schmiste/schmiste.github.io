1. INSTALLATION

To install our CVSAP solver, extract the contents of cvsap-solver.tar.gz to any directory. Now download and install the SCIP optsuite (see http://scip.zib.de/download.shtml). Our implementation is tested with versions 3.0.0 and 3.0.1 of the SCIP optsuite. Create an empty folder "lib" in the directory where you unpacked cvsap-solver.tar.gz and create a symlink on scip:

tar -zxvf cvsap-solver.tar.gz mkdir cvsap-solver/lib ln -s XYZ/scipoptsuite-3.0.1/scip-3.0.1 cvsap-solver/lib/scip

where XYZ is the directory in which you installed scipoptsuite. Note that the above tar command will create a folder cvsap-solver/ in the current directory.

2. COMPILATION

To compile our solver, just type make in the directory cvsap-solver/ (or the director in which you unpacked the contents of cvsap-solver.tar.gz).

3. USAGE

If you successfully compiled the solver using make, the directory bin\ should contain the executable cvsap. To solve an A-CVSAP instance just use

cvsap <instance>

where instance is a data file as specified in the file cvsap-solver/DATAFORMAT. Note that you can supply further parameters to our solver by means of a settings file that is statically named cvsap.set and must be contained in the directory you start the executable cvsap from. An overview over the possible parameters that may be stated in cvsap.set is given in the file cvsap-solver/PARAMETERS. To solve an instance of our grid experiments navigate e.g. to the directory grid_16x16 and type

XYZ/bin/cvsap grid_16x16_1.scip

4. PROBLEMS

If you experience any problems while installing or compiling our CVSAP solver, please feel free to contact Matthias Rost (mrost@net.t-labs.tu-berlin.de) for guidance.
